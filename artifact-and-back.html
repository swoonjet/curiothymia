One small step first

This app needs a local server to fetch images from museum and library collections. The browser blocks those requests when opening the file directly.

Double-click the launcher in your folder:

start-artifact-and-back.command

Or run this in Terminal from the folder:

python3 -m http.server 8899

Then open http://localhost:8899/artifact-and-back.html
